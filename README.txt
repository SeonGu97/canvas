클래스를 사용해서 객체화 시켰다. 몇몇의 예제들을 찾아보니까 공을 만들고
튕기게 만드는 방법이 조금은 달랐다. 그냥 코드로 짜서 만드는 것도 있었고,
뭔가 깔끔하게 객체화 시켜서 만드는 법도 있었는데, 개인적으로 클래스를 사용해서
만드는게 조금더 맘에 들었다. 다음은 벽을 만들어서, 공이 벽에도 닿으면 튕겨나가도록
만들어 봐야겠다.

(21.10.13)
- 어제는 지금 만드는 프로젝트를 타입스크립트를 사용해서 만들어 볼려고 유튜브 강의를 들으면서
기초를 익혔는데, 중간에 exports의 오류로 인해서 타입스크립트로 만들어 놓고, 막상 구동은 
되지 않았다. 그래서 해결방법으로 webpack을 사용하면 된다고 하길래 오늘 유튜브 강의를
보았는데, 사용하는 방법은 나와 있었지만, 이 문제를 해결하는 방안은 없었다. 그래서 타입스크립트
의 사용을 중단하고 자바스크립트로 다시 만들기 시작했다. 오늘은 공이 튕기는 가운데에
블록을 설치해서 블록에 닿으면 튕겨 나가도록 만들었다. 유튜브를 보면서 따라 만들었는데,
새로운 문법도 접했다. canvas를 사용하니까 기존에 프로젝트를 만들떄와는 다른 메소드를 사용해서
익숙하지는 않다. 계속해서 코드를 다시 읽어보면서 완벽하게 숙지를 해야겠다.
